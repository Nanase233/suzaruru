# 🎨 Blind Spot Mirror – 绘画盲点检测工具

📄 产品概念单页 (The One-Pager)

## 产品名称

**Blind Spot Mirror**

------

## 一句话 Slogan

**画画太久会“眼盲”？让AI帮你发现画面里的隐藏问题。**

------

## 用户痛点

**视觉适应问题：**
 画师长时间盯着同一张画，容易产生“视觉适应效应”，导致一些明显的问题反而看不出来，例如：

- 人物五官微妙错位
- 画面整体倾斜
- 构图重心偏移
- 明暗对比不均

**检查方式低效：**

画师通常用这些方法检查画面：

- 手动翻转画布
- 缩小画面远距离观察
- 发给朋友帮忙看

这些方法 **效率低、没有量化指标**。

**真实场景：**

一位插画师完成角色立绘后发到社交平台，评论区第一条：

> “脸有点歪。”

画师重新看才发现 **眼睛位置确实不对称**，但自己已经看了3小时完全没发现。

------

## 解决方案 (Solution)

**Blind Spot Mirror** 是一款 **数字绘画视觉分析工具**，通过计算机视觉帮助画师发现自己忽略的画面问题。

它不会评价艺术好坏，只提供 **客观视觉数据**。

### 核心功能

| 功能           | 作用                  |
| -------------- | --------------------- |
| ⚖️ 构图平衡检测 | 分析画面左右视觉重量  |
| 📐 倾斜检测     | 判断画面是否歪斜      |
| 🪞 镜像检查     | 一键翻转画布          |
| 🧍 对称性检测   | 检查人物脸部/物体结构 |
| 🌗 明暗分布分析 | 可视化亮度与对比度    |

软件以 **悬浮辅助窗口** 运行，可以在 **Photoshop / Clip Studio / Krita** 旁边使用。

------

## 使用的 AI 技术 (AI Technology)

- **计算机视觉 (Computer Vision)**
  - 图像重心计算
  - 对称性检测
  - 边缘结构识别
  - 明暗分布分析
- **图像处理算法**
  - OpenCV
  - 边缘检测 / 轮廓检测

本产品 **不会生成绘画内容，仅提供视觉分析辅助**。
